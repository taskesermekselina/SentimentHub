## 🚀 SentimentHub Nedir?

SentimentHub, e-ticaret platformlarında yer alan kullanıcı yorumlarını yapay zekâ teknolojileri kullanarak analiz eden ve elde edilen verileri karar destek bilgilerine dönüştüren bir iş zekâsı platformudur.

Sistem yalnızca yorumların olumlu veya olumsuz olduğunu belirlemekle kalmaz; ürün kalitesi, fiyat algısı, teslimat deneyimi, satıcı hizmetleri ve genel müşteri memnuniyeti gibi kritik boyutları ayrı ayrı değerlendirir.

Geliştirilen AnalysisEnrichmentService katmanı sayesinde kök neden analizi, risk erken uyarı sistemi, fiyat algısı, müşteri sadakati ve tavsiye etme eğilimi gibi ileri düzey içgörüler üretilebilmektedir.

Karşılaştırmalı İş Zekâsı Modülü sayesinde farklı ürünler ortak bir değerlendirme çerçevesinde analiz edilerek ürün performansları karşılaştırılabilmektedir.

## 🧠 Yapay Zekâ Analiz Süreci

SentimentHub, kullanıcı yorumlarını çok aşamalı bir yapay zekâ işleme hattı üzerinden analiz etmektedir.

```text
Trendyol Ürün Linki
        ↓
Yorumların Toplanması
        ↓
Veri Temizleme ve Ön İşleme
        ↓
LLM Tabanlı Duygu Analizi
        ↓
Boyut Tabanlı Analiz
        ↓
AnalysisEnrichmentService
        ↓
Karşılaştırmalı İş Zekâsı Motoru
        ↓
Dashboard ve PDF Raporlama
```

Sistem ilk olarak ürün yorumlarını toplar ve tekrar eden kayıtları temizler. Ardından yorumlar yerel büyük dil modeli kullanılarak analiz edilir. Her yorum duygu durumu ve ilgili kategori açısından değerlendirilir.

Elde edilen sonuçlar AnalysisEnrichmentService katmanına aktarılır ve burada ileri düzey iş zekâsı analizleri gerçekleştirilir. Son aşamada tüm sonuçlar görselleştirilir ve raporlanır.

---

## 🚀 AnalysisEnrichmentService

AnalysisEnrichmentService, temel duygu analizi sonuçlarını stratejik iş zekâsı çıktılarına dönüştüren gelişmiş analiz katmanıdır.

### 🔍 Kök Neden Analizi

Olumsuz müşteri deneyimlerine neden olan temel problemleri otomatik olarak belirler.

- Kalite problemleri
- Teslimat sorunları
- Paketleme şikayetleri
- Satıcı kaynaklı problemler
- Fiyat memnuniyetsizliği

gibi tekrar eden temaları tespit eder.

---

### ⚠️ Risk Erken Uyarı Sistemi

Mevcut analiz sonuçlarını geçmiş raporlarla karşılaştırarak müşteri memnuniyetindeki değişimleri takip eder.

Risk seviyeleri:

- 🟢 Düşük Risk
- 🟡 Orta Risk
- 🟠 Yüksek Risk
- 🔴 Kritik Risk

olarak sınıflandırılır.

---

### 💰 Fiyat Algısı Analizi

Kullanıcıların ürünü ekonomik açıdan nasıl değerlendirdiğini analiz eder.

İncelenen örnek ifadeler:

- Fiyat performans ürünü
- Parasını hak ediyor
- Bu fiyata değmez
- Çok pahalı

---

### 📢 Tavsiye Etme Eğilimi Analizi

Yorum içeriklerinden kullanıcıların ürünü başkalarına önerme potansiyeli hesaplanır.

Sonuçlar:

- Yüksek Potansiyel
- Orta Potansiyel
- Düşük Potansiyel

olarak raporlanır.

---

### ❤️ Müşteri Sadakati Analizi

Tekrar satın alma davranışını ve marka bağlılığını gösteren yorumlar analiz edilir.

Örnek sadakat sinyalleri:

- İkinci siparişim
- Tekrar satın aldım
- Favori markam oldu
- Bundan sonra hep buradan alacağım

---

## 🏆 Karşılaştırmalı İş Zekâsı Motoru

SentimentHub'ın en güçlü modüllerinden biri Karşılaştırmalı İş Zekâsı Motoru'dur.

Bu modül sayesinde farklı ürünler ortak bir değerlendirme çerçevesinde analiz edilerek performans karşılaştırmaları gerçekleştirilmektedir.

### Üretilen Çıktılar

| Çıktı | Açıklama |
|---------|---------|
| 🏆 Kazanan Ürün | Karşılaştırmayı kazanan ürünü belirler |
| 📊 Karar Güven Skoru | Sonucun güvenilirlik seviyesini hesaplar |
| ⚖️ Kategori Dominansı | Hangi kategorilerde üstünlük olduğunu gösterir |
| 😊 Duygu Tutarlılığı | Duygu dağılımlarını değerlendirir |
| 👤 Persona Uyumu | Hedef kullanıcı profillerini belirler |
| 🔄 Geçiş Öngörüsü | Kullanıcı geçiş avantajlarını hesaplar |
| 📝 Karar Gerekçesi | Sonucun neden oluştuğunu açıklar |

## ⚙️ Kurulum

### Gereksinimler

- .NET 8 SDK
- Python 3.11+
- LM Studio
- Google Chrome
- SQLite veya SQL Server

### Depoyu Klonlayın

```bash
git clone https://github.com/taskesermekselina/SentimentHub.git
```

### Web Uygulamasını Çalıştırın

```bash
dotnet run
```

### Yapay Zekâ Servisini Başlatın

```bash
uvicorn main:app --reload
```

### LM Studio

Yerel büyük dil modelinin çalıştırılması ve API servisinin aktif hale getirilmesi gerekmektedir.

Varsayılan API adresi:

```text
http://localhost:1234/v1
```
